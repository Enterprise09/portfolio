# Portfolio
> Enterprise09's portfolio page

## Construct

- Create project with "npx create-react-app" command

## Start project

```shell

    git clone https://github.com/Enterprise09/portfolio.git
    cd portfolio

    # open with vscode
    code .

    # install dependency
    npm install

    # start project by port 3000
    npm start

```

- check "localhost:3000/" at browser
